On this website you can also find example files for downloading and testing. Here is the documentation of my test run:

### CONIFER ######################################
# Create RPKM files
python /users/GD/tools/conifer/conifer_v0.2.2/conifer.py rpkm --probes probes.txt --input sample.bam --output RPKM/sample.rpkm.txt

# Main CoNIFER analysis:
python /users/GD/tools/conifer/conifer_v0.2.2/conifer.py analyze --probes probes.txt --rpkm_dir ./RPKM/ --output analysis.hdf5 --svd 6 --write_svals singular_values.txt --plot_scree screeplot.png --write_sd sd_values.txt

# Making calls:
python /users/GD/tools/conifer/conifer_v0.2.2/conifer.py call --input analysis.hdf5 --output calls.txt

# Exporting data for segmentation and viewing:
mkdir export
python /users/GD/tools/conifer/conifer_v0.2.2/conifer.py export --input analysis.hdf5 --output ./export/

# Exporting data for segmentation and viewing for only one sample:
python /users/GD/tools/conifer/conifer_v0.2.2/conifer.py export --input analysis.hdf5 --sample SampleID --output ./export/

# Printing and viewing CoNIFER data for one sample:
python /users/GD/tools/conifer/conifer_v0.2.2/conifer.py plot --input analysis.hdf5 --region chr#:start-stop --output image.png --sample sampleID
(optional with multiple samples: --sample r:sampleID1 b:sampleID2 g:sampleID3)

# Printing and viewing CoNIFER data for all one samples:
mkdir call_imgs
python /users/GD/tools/conifer/conifer_v0.2.2/conifer.py plotcalls --input analysis.hdf5 --calls calls.txt --outputdir ./call_imgs/
